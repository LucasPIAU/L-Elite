Comment résoudre le CAPTCHA ?

Voici comment résoudre le CAPTCHA :

  • Un mot aléatoire est affiché, et vous devez l'écrire à partir du hash généré.
  • Le hash est une chaîne de caractères que vous pouvez couper en portions de taille correspondante au chiffre du compteur lors de son arrêt.
  • Vous devez collecter ces portions et les placer dans l'ordre correct dans la zone de reconstruction.
  • Utilisez le bouton "Stop" pour stopper le compteur puis sélectionnez une portion du hash en cliquant dessus avec la souris.
  • Lorsque vous pensez avoir reconstruit le mot, cliquez sur "Vérifier".
